Application Introduction

This application allows users to add/update/delete customers and their corresponding appointments to a table.
Users can also view three types of reports:
- the total number of customer appointments by type and month
- a schedule for each contact in your organization that includes appointment ID, title, type and description, start date and time, end date and time, and customer ID.
- The last report displays the total number of customers per division name(state).




How to run the application

1. Make sure the IntelliJ, JDK, and JavaFX version is up to date and the same as the provided information above.
2. Unzip the application file and open it in IntelliJ.
3. Build the project and run the application
